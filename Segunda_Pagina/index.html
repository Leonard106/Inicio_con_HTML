Formatos de Textos

En esta página podremos observar algunos de los diferentes formatos de texto que podemos generar a traves de las diferentes etiquetas.

Para genenrar un texto con negritas usamos la etiqueta "b"

Negritas

Para generar texto en cursiva usaremos la etiqueta "i"

Cursivas

Para generar texto tachado o eliminado usamos la etiqueta "del"

Tachado

Para generar texto mas pequeño usamos la etiqueta "small"

Pequeño

Para generar texto Subrayado usamos la etiqueta "mark"

Importante
